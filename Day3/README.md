# Day 3 Task 
## ChatApp Project (TypeScript)

A simple command-line chat application built with TypeScript.

## Features
- Send messages and receive bot replies
- Delete messages by ID
- Exit the application
- In-memory message storage

## Technologies
- TypeScript
- Node.js

## Run

```bash
npm install
npm run build
npm start
```

## Changes Made
- Converted JavaScript to TypeScript
- Added interfaces (`ChatMessage`, `ChatResponse`)
- Added type annotations
- Preserved the original functionality
